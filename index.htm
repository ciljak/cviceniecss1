About layout

Layaout is a way how content of a webpage (elements of a page, paragraphs, headlines, images and so on) flow through browser window.

There are these types of webpage layouts:

  1. Fixed layout - hose that use a specific unit of measurement to define web page widths – independent of all other factors, including browser window dimensions and font sizes. In other words, the webpage/site will be the same fixed sized regardless of the size of the browser window or the size of font in which the page is written.
  2. Relative layouts - defines webpage width using some relative unit of measurement. In this case, the content in the page will resize according to the size of browser window users access it on.
